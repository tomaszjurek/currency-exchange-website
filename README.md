# currency-exchange-website
Website which will take data from NBP API. There will be a exchange calculator too.  

## Getting Started
```
git clone https://github.com/tomaszjurek/currency-exchange-website.git
```

### Running
```
open index.html file.
```

## Built With
* HTML 5
* CSS 3
* SCSS
* JavaScript ES6
* Bootstrap 4
* Webpack
* Gulp
